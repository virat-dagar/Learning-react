## React Features

### 1. Virtual DOM:

- This is a lightweight representation of real DOM.
- There are 2 virtual DOMs
1. Copy of the real DOM
2. Manages the changes

- when 2 virtual DOMs get compared with each other, this process
  is known as DIFFING ALGORITHM
- and finally, when changes are finalised, the copy
  of the real DOM(V1) updates changes in the real DOM.
  And this process is called RECOINCILIATION
